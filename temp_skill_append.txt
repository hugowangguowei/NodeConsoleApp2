

## 6. Buff 引用表 (Buff Reference Table)

| **狂暴姿态 (Berserk)** | `buff_berserk` | - |

| **尖刺防御 (Spiked Shell)** | `buff_thorns` | - |
| **反击姿态 (Counter Stance)** | `buff_counter` | - |

| **盾墙 (Shield Wall)** | `buff_shield_wall` | - |
