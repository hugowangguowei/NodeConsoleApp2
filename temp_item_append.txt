

## 6. Buff 引用表 (Buff Reference Table)

以下列出了装备与 Buff 的对应关系。大部分装备的特殊效果通过被动 Buff 实现。

| 装备 ID | 对应 Buff ID | 说明 |
| :--- | :--- | :--- |
| `wp_knight_sword` | `passive_knight` | 造成伤害增加 |
| `wp_vampire_fang` | `passive_vampire` | 攻击吸血 |
| `wp_thunder_hammer` | `passive_thunder_strike` | 攻击附带雷电追伤 |
| `wp_cursed_blade` | `passive_cursed_self_dmg` | 每回合对自己造成伤害 |
| `body_chainmail` | `passive_heavy_armor` | 速度降低 |
| `helm_gladiator` | `debuff_start_weak` | 首回合 AP 减少或伤害降低 |
| `body_dragon_scale`| `passive_dragon_scale` | 受到的伤害减少 |
| `acc_amulet_focus` | `buff_ap_recovery` | 概率或条件回复 AP |
| `acc_boots_speed` | `passive_speed_boost` | 速度/闪避提升 |
| `acc_pendant_phoenix`| `passive_phoenix` | 战斗复活一次 |
